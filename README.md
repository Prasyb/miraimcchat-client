# MiraiMcChat-client
![Github](https://img.shields.io/badge/Author-Prasyb-blue)  

配合 [MiraiMcChat]() 的Mod，提供QQ群与Minecraft的聊天同步功能

本Mod为客户端，需配合mirai-console插件 [MiraiMcChat]() 使用

## 使用方法

### 1.构建服务端

- **[MiraiMcChat]()**

### 2.连接服务端

- 在聊天栏或控制台使用 `/chatconnect <host:port> <key>` 连接服务端，其中 key 在注册客户端时取得
